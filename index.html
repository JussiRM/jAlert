How to Use

Grab the JS File here and include it somewhere on the page

Use by calling jAlert('text', { options });

Options

time: time in milliseconds how long the popup should be seen (default: 2000)
vAlign: 'top', 'middle', or 'bottom'. Where the popup is vertically placed. (default: 'bottom')
useCssAnimation: If you want to use CSS Animations (smoother), true. (default: false)

Eg.
jAlert('this is some text', { time: 1000, vAlign: 'middle', useCssAnimation: true });
